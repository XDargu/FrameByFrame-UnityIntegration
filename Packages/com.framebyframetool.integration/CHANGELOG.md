# Changelog

All notable changes to Frame by Frame for Unity are documented here.

## [1.0.0] - 2026-07-08

### Added

- First production-ready Unity Package Manager release.
- Package identity `com.framebyframetool.integration`.
- Support for Frame by Frame v1.1.0.
- Editor window under **Window > Frame by Frame**.
- Project Settings under **Project Settings > Frame by Frame**.
- Runtime recording option registry using `FrameByFrameRecordingOption`.
- Build-safe runtime settings asset for internal/development builds.
- Built-in recorder components for physics, colliders, NavMesh, NavMesh agents, animation, logs, helper planes, and screenshots.
- Demo sample package.
- Documentation for quick start, recording options, built-in recorders, API usage, examples, and internal builds.
